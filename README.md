# Great Proxy

A fast, ad-free web proxy you can deploy to Vercel.
Just enter any URL or domain (like `github.com`, `crazygames.com`) and it will auto-prepend `https://`, proxy the site, and eliminate basic ads.

## Usage
1. Clone this repo or download the files.
2. Deploy to [Vercel](https://vercel.com/) (create a new project and import your repo or drag and drop your folder).
3. Visit your deployed app and enjoy browsing ad-free via the proxy.

## Features
- Accepts bare domains and full URLs.
- Simple ad removal (strips `<script>` and `<iframe>` tags).
- No ads, fast, simple UI.
- Deploys quickly on Vercel.
